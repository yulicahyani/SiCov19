# SiCov19

Submission Belajar Fundamental Front-End Web Development (Dicoding)

**CARA INSTALASI**

 1. Menginstall package yang digunakan

    npm install

 2. Menjalankan aplikasi menggunakan webpack production

    npm run build

 3. Atau menggunakan webpack development
 
    npm run start-dev
